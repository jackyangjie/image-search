## 1. 用户故事概述

### 1.1 什么是用户故事

用户故事是从用户视角描述需求的简短陈述，格式为：

> **作为** [角色]，**我想要** [功能]，**以便** [价值/目的]

### 1.2 用户角色定义

| 角色代码 | 角色名称 | 描述 |
|---------|---------|------|
| U1 | 摄影爱好者 | 拥有大量照片(1万+)，需要按场景快速查找 |
| U2 | 宝妈/宝爸 | 大量孩子照片，需要按时间/场景/表情查找 |
| U3 | 商务人士 | 截图、文档照片多，需要快速检索 |
| U4 | 隐私敏感用户 | 拒绝云同步，要求完全本地处理 |
| U5 | 首次使用用户 | 对应用不熟悉，需要引导 |

### 1.3 故事优先级

| 优先级 | 说明 | 时间框架 |
|--------|------|---------|
| P0 (Must Have) | 必须实现，MVP核心功能 | 第1-2周 |
| P1 (Should Have) | 应该实现，重要功能 | 第3-4周 |
| P2 (Could Have) | 可以实现，增强体验 | 第5-8周 |
| P3 (Won't Have) | 暂不实现，未来版本 | v2.0+ |

## 2. 核心功能用户故事

### 2.1 自然语言搜索 (P0)

#### US-001: 基本自然语言搜索
**作为** U1摄影爱好者，  
**我想要** 通过输入自然语言描述来搜索照片，  
**以便** 快速找到特定场景的照片而不需要手动浏览数千张照片。

**验收标准**:
- [ ] 可以输入"海边的日落"并找到匹配照片
- [ ] 可以输入"穿红衣服的人在公园"并找到匹配照片
- [ ] 搜索响应时间 < 500ms
- [ ] 支持中文输入

#### US-002: 时间维度搜索
**作为** U2宝妈，  
**我想要** 在搜索中包含时间信息（如"去年春节"），  
**以便** 快速找到特定时间段的照片。

**验收标准**:
- [ ] 支持"去年"、"上个月"、"上周"等相对时间
- [ ] 支持"2024年春节"、"去年夏天"等具体时间段
- [ ] 时间和场景可以组合搜索

#### US-003: 搜索结果排序
**作为** U1摄影爱好者，  
**我想要** 搜索结果按相关性排序，  
**以便** 最匹配的照片排在最前面。

**验收标准**:
- [ ] 相似度最高的照片排在首位
- [ ] 显示相似度分数（可选）
- [ ] 支持按时间重新排序

### 2.2 相册扫描与索引 (P0)

#### US-004: 首次启动自动扫描
**作为** U5首次使用用户，  
**我想要** 应用首次启动时自动扫描我的相册，  
**以便** 快速建立照片索引开始搜索。

**验收标准**:
- [ ] 首次启动请求相册权限
- [ ] 权限同意后自动开始扫描
- [ ] 显示扫描进度
- [ ] 扫描完成前可以使用已索引的照片

#### US-005: 后台增量更新
**作为** U1摄影爱好者，  
**我想要** 新增照片自动被索引，  
**以便** 不需要手动触发扫描也能搜索到新照片。

**验收标准**:
- [ ] 新增照片自动检测
- [ ] 后台自动索引新照片
- [ ] 索引完成通知（可选）

#### US-006: 扫描进度显示
**作为** U1摄影爱好者（拥有1万+照片），  
**我想要** 看到详细的扫描进度，  
**以便** 了解索引何时完成。

**验收标准**:
- [ ] 显示已扫描/总照片数
- [ ] 显示预计剩余时间
- [ ] 显示当前处理的照片
- [ ] 支持暂停/恢复扫描

### 2.3 照片浏览 (P0)

#### US-007: 网格浏览照片
**作为** U2宝妈，  
**我想要** 以网格形式浏览所有已索引照片，  
**以便** 快速查看照片缩略图。

**验收标准**:
- [ ] 照片按时间倒序排列
- [ ] 显示缩略图
- [ ] 支持下拉刷新
- [ ] 分页加载，避免卡顿

#### US-008: 查看大图
**作为** U3商务人士，  
**我想要** 点击照片查看高清大图，  
**以便** 确认照片内容细节。

**验收标准**:
- [ ] 点击照片进入大图模式
- [ ] 支持手势缩放
- [ ] 支持左右滑动切换照片
- [ ] 显示照片元信息（时间、大小等）

### 2.4 隐私与安全 (P0)

#### US-009: 完全本地处理
**作为** U4隐私敏感用户，  
**我想要** 所有AI处理和照片数据完全在本地，  
**以便** 我的照片不会上传到任何云端服务器。

**验收标准**:
- [ ] 无需网络权限即可使用核心功能
- [ ] 无需用户登录
- [ ] 照片向量数据本地存储
-- [ ] 应用不发送照片到外部服务器

#### US-010: 应用锁
**作为** U4隐私敏感用户，  
**我想要** 设置应用锁（指纹/面容/密码），  
**以便** 防止他人访问我的照片。

**验收标准**:
- [ ] 支持系统级生物识别
- [ ] 支持数字密码
- [ ] 可以选择是否启用
- [ ] 后台切换回来时验证

## 3. 用户画像详细故事

### 3.1 摄影爱好者 (U1) - 小明

**背景**: 小明是一名业余摄影师，手机里有2万多张照片，包括风景、人像、街拍等。他经常需要找特定场景的照片来发社交媒体或参加比赛。

#### US-101: 按场景快速查找
**作为** 小明，  
**我想要** 搜索"银杏大道秋天"找到去年在钓鱼台拍的银杏照片，  
**以便** 参加"最美秋天"摄影比赛。

**场景描述**:  
小明去年秋天在钓鱼台银杏大道拍了很多照片，但散落在2万多张照片里。他需要找到其中光线最好、构图最美的那几张参加比赛。

**验收标准**:
- [ ] 支持"银杏"、"秋天"、"大道"等关键词组合
- [ ] 搜索结果包含时间和地点信息
- [ ] 可以按时间筛选（2024年10-11月）
- [ ] 支持相似度阈值调整

#### US-102: 批量导出搜索结果
**作为** 小明，  
**我想要** 将搜索结果批量导出到特定相册，  
**以便** 整理参赛照片。

**验收标准**:
- [ ] 支持多选搜索结果
- [ ] 可以导出到系统相册
- [ ] 导出进度显示

### 3.2 宝妈 (U2) - 李女士

**背景**: 李女士是一位2岁孩子的妈妈，手机里全是孩子的成长照片，按时间顺序排列。她经常想找孩子特定表情或活动的照片分享给家人。

#### US-201: 找宝宝第一次
**作为** 李女士，  
**我想要** 搜索"宝宝第一次走路"找到那些珍贵瞬间，  
**以便** 制作成长纪念册。

**场景描述**:  
李女士记得孩子第一次走路是在客厅，大概在今年3月。她想找到那几张照片，但当时没来得及打标签，现在要从几千张照片里找出来。

**验收标准**:
- [ ] 支持"宝宝"、"走路"、"第一次"等概念搜索
- [ ] 支持按时间范围筛选（2024年3月）
- [ ] 可以识别室内/室外场景
- [ ] 搜索结果包含人物检测

#### US-202: 找特定表情的照片
**作为** 李女士，  
**我想要** 搜索"宝宝笑"找到孩子开心的照片，  
**以便** 发给爷爷奶奶看。

**验收标准**:
- [ ] 支持表情相关描述（笑、哭、惊讶等）
- [ ] 可以识别人脸表情
- [ ] 搜索结果按时间排序

### 3.3 商务人士 (U3) - 张经理

**背景**: 张经理经常参加会议、培训，拍摄白板、PPT、文档照片。他需要快速找到某次会议的资料。

#### US-301: 找会议白板照片
**作为** 张经理，  
**我想要** 搜索"上周的会议白板"快速找到需要的资料，  
**以便** 整理会议纪要。

**场景描述**:  
张经理上周参加了3个会议，每个会议都拍了白板照片。现在要整理其中一个会议的笔记，但照片混在一起，需要快速筛选。

**验收标准**:
- [ ] 支持"上周"、"会议"、"白板"等组合搜索
- [ ] 可以识别白板/屏幕内容
- [ ] 显示拍摄时间和地点

#### US-302: 找文档照片
**作为** 张经理，  
**我想要** 搜索"合同扫描件"找到之前拍的文档照片，  
**以便** 查看合同内容。

**验收标准**:
- [ ] 支持文档类型识别
- [ ] 可以搜索文字内容（OCR预留）
- [ ] 支持文档/照片分类筛选

### 3.4 隐私敏感用户 (U4) - 王先生

**背景**: 王先生是一名律师，手机里有很多敏感的客户资料照片。他完全不相信云存储，要求所有处理都在本地。

#### US-401: 无需网络权限
**作为** 王先生，  
**我想要** 在安装时看到应用不需要网络权限，  
**以便** 确认照片不会外泄。

**场景描述**:  
王先生在安装任何应用前都会检查权限。他希望看到这个应用只需要存储权限，不需要网络、位置等其他权限。

**验收标准**:
- [ ] 应用在安装时只申请必要权限（存储、相册）
- [ ] 不需要网络权限声明
- [ ] 隐私政策明确说明数据本地处理
- [ ] 首次启动展示隐私承诺

#### US-402: 完全离线使用
**作为** 王先生，  
**我想要** 在飞行模式下也能搜索我的照片，  
**以便** 确保所有功能不依赖网络。

**验收标准**:
- [ ] 开启飞行模式后所有功能正常使用
- [ ] 搜索、浏览、扫描都正常工作
- [ ] 应用不尝试网络连接

## 4. 验收标准

### 4.1 功能验收标准

#### 搜索功能
| 验收项 | 标准 | 测试方法 |
|--------|------|---------|
| 搜索响应时间 | < 500ms | 性能测试 |
| 搜索准确率 | Top-5 准确率 > 80% | 人工标注测试集 |
| 支持语言 | 中文、英文 | 功能测试 |
| 输入长度 | 支持100字符 | 边界测试 |

#### 扫描功能
| 验收项 | 标准 | 测试方法 |
|--------|------|---------|
| 扫描速度 | > 10张/秒（中端手机） | 性能测试 |
| 增量更新 | 新照片自动检测 | 功能测试 |
| 断点续传 | 支持暂停/恢复 | 功能测试 |
| 存储占用 | 原始照片的5-10% | 存储测试 |

#### 隐私功能
| 验收项 | 标准 | 测试方法 |
|--------|------|---------|
| 网络流量 | 核心功能零网络请求 | 抓包测试 |
| 权限申请 | 仅申请必要权限 | 权限检查 |
| 数据存储 | 完全本地存储 | 文件系统检查 |

### 4.2 性能验收标准

| 指标 | 目标值 | 测试设备 |
|------|--------|---------|
| 冷启动时间 | < 3秒 | 中端手机 |
| 搜索响应时间 | < 500ms | 中端手机 |
| 索引速度 | > 10张/秒 | 中端手机 |
| 内存占用 | < 200MB | 中端手机 |
| 存储占用 | 原始照片的5-10% | - |

| 平台 | 最低版本 | 测试设备 |
|------|---------|---------|
